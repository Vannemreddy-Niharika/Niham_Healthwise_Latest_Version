## Step 1: Create GitHub Repository (30 seconds)

1. Click this link: **https://github.com/new**
2. In the "Repository name" box, type: `niham-healthwise`
3. Click the green **"Create repository"** button at the bottom

That's it for Step 1! ‚úÖ

---

## Step 2: Copy Your Repository URL (10 seconds)

After creating the repository, you'll see a page with commands.

Look for a URL that looks like this:
```
https://github.com/YOUR_USERNAME/niham-healthwise.git
```

**Copy that entire URL!** (Click the copy button next to it)

---

## Step 3: Run These Commands (1 minute)

Open **Command Prompt** (Windows) or **Terminal** (Mac/Linux) in your project folder.

Then paste these commands **ONE BY ONE**:

### Command 1:
```bash
git remote add origin PASTE_YOUR_URL_HERE
```
**Replace `PASTE_YOUR_URL_HERE` with the URL you copied!**

Example:
```bash
git remote add origin https://github.com/john/niham-healthwise.git
```

### Command 2:
```bash
git branch -M main
```

### Command 3:
```bash
git push -u origin main
```

**Done!** Your code is now on GitHub! üéâ

---

## Step 4: Enable Free Hosting (30 seconds)

1. Go to your repository on GitHub
2. Click **"Settings"** (top menu)
3. Click **"Pages"** (left sidebar)
4. Under "Branch", select **"main"**
5. Click **"Save"**

Wait 2 minutes, then your site will be live at:
```
https://YOUR_USERNAME.github.io/niham-healthwise/
```
